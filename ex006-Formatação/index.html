Principais Formatações

Negrito / Destaque

Termo em negrito com a tag <b>.

Termo em destaque com a tag <strong>.

Itálico / Ênfase

Termo em itálico com a tag <i>.

Termo em ênfase com a tag <em>.

Texto Marcado

Termo marcado com a tag <mark>.

Grande / Pequeno

Termo grande com a tag <big>.

Termo pequeno com a tag <small>.

Texto Excluído/Deletado

Termo excluído com a tag <del>.

Texto Incluído/Inserido / Sublinhado

Termo inserido com a tag <ins>.

Termo sublinhado com a tag <u>.

Texto Sobrescrito

Termo sobrescrito com a tag <sup>.

Exemplo: x20+3

Texto Subscrito

Termo subrescrito com a tag <sub>.

Exemplo: H2O

Código-fonte

Termo de mono espaçado usando a tag <code>.

Indentação

Exemplo: 

            
num = int(input('Digite um número: '))
if num % 2 == 0:
    print(f'O número {num} é par.')
else:
    print(f'O número {num} é ímpar.')

Acima, código usando a tag <code> indentado usando a tag <pre>.
